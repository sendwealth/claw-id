# 贡献指南

感谢你考虑为 CLAW ID 做贡献！

---

## 🎯 贡献方式

### 1. 报告 Bug

在 [GitHub Issues](https://github.com/sendwealth/claw-id/issues) 中提交：

- 描述问题
- 复现步骤
- 预期行为
- 实际行为
- 环境信息

### 2. 提交功能请求

在 Issues 中标记为 `enhancement`：

- 描述功能
- 使用场景
- 预期效果

### 3. 提交代码

1. Fork 仓库
2. 创建分支 (`git checkout -b feature/amazing`)
3. 提交代码 (`git commit -m 'Add amazing feature'`)
4. 推送分支 (`git push origin feature/amazing`)
5. 创建 Pull Request

---

## 📋 代码规范

### JavaScript

- 使用 ES6+ 语法
- 遵循 Airbnb 风格指南
- 添加必要注释
- 编写单元测试

### Commit 消息

```
feat: 添加新功能
fix: 修复 Bug
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试相关
chore: 构建/工具
```

---

## ✅ Pull Request 检查清单
