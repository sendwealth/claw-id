## 📝 实现细节

### 缓存策略

1. **用户信息缓存** (5分钟TTL)
   ```javascript
   cacheKey: `user:${userId}`
   缓存完整用户对象(不含密码)
   ```

2. **Agent列表缓存** (2分钟TTL)
   ```javascript
   cacheKey: `agents:${userId}:${page}:${limit}`
   仅缓存基础查询(无过滤条件)
   ```

3. **缓存失效**
   - 自动过期 (TTL机制)
   - 数据更新时需要手动清除缓存

### Redis配置 (可选)

启用Redis缓存以获得更好的性能:

```bash
# 在 .env 文件中添加
REDIS_URL=redis://localhost:6379
```

不配置REDIS_URL时,系统会自动使用内存缓存(降级模式)。

### 压缩配置

```javascript
compression({
  threshold: 1024,  // >1KB才压缩
  level: 6          // 压缩级别 (1-9)
})
```
